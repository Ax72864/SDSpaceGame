# v0.20.0-mining-coverage-path-evidence PM 看板

## 本轮目标

版本名：`v0.20.0-mining-coverage-path-evidence`。

本轮目标是在不发布、不改核心规则、不改资源布局的前提下，补齐采矿覆盖路径与收益归因证据：证明玩家能否从原路径或明确替代路径让矿站覆盖资源外环，并看到 `harvestingCount > 0` 与可归因收益；若现有规则路径不可达，则通过只读诊断和必要最小反馈让玩家理解“当前位置采不到 / 需要重新选址或靠近资源外环”。

## 背景

- 制作人方向文档：`Workspaces/game-producer-lead-designer/v0.20.0-mining-coverage-path-evidence-direction.md`。
- v0.19 已关闭维修链误读：`repair` 在原路径建成，资源不足提示已明确为“下一座还缺”，不是刚才建造失败。
- v0.19 采矿不再判定为“不产出 bug”，而是明确收敛为路径覆盖问题：`mining=-1,1` 已建成且 active，但未覆盖资源外环。
- 当前关键阻塞：`harvestingCount=0`、`target=null`、`allOutOfRange=true`，等待窗口内资源增长不能归因于采矿收益。
- 自然交火、战后复盘和 Meta 回流仍阻塞于采矿覆盖 / 收益证据未闭环。
- 本阶段是内部证据阶段，不是发布阶段，不进入 tag、构建或 `Releases/` 归档。

## 风险定级

- 风险等级：中风险。
- 中风险前提：仅做采矿覆盖路径证据、只读路径 / 距离诊断、PM 记录和必要最小反馈修复，不改核心规则与发布资产。
- 高风险升级触发：若触及采矿范围 / 公式、资源布局、移动能力、资源生成公式、建造合法性、存档、`SAVE_KEY`、meta schema、依赖安装、发布、tag 或 `Releases/` 归档，立即暂停并升级高风险流程或等待用户确认。
