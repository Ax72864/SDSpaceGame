## 实现范围

### A 金属 / 成本缺口诊断

- `computeMiningExpansionAffordability()`：基于 `bestCandidate` + `suggestedExpansionPath` 计算框架步数、采矿站成本、合计 metal/ore 需求。
- `computeMiningCoverageDiagnostics()` 输出新增 `affordability` 字段，保留 v0.20 既有字段。
- `buildMiningExpansionGuidanceHtml()` / `buildResourceGuideHtml()` 展示「路径可支付 / 建议路径可达但付不起」与缺口摘要。

### E 候选路径 / ghost path

- `buildMiningExpansionGuidanceHtml()`：最佳候选格、框架步数、路径 grid key 链（桌面完整、语义区分建议 vs 可支付）。
- `renderMiningExpansionGhostPath()`：canvas 半透明框架步 ring + 目标格虚线 ring（未采集中时显示）。
- 诊断输出 `ghostPath`：`targetGridKey`、`stepGridKeys`、`frameCount`、`finalAction`。
- 测试 hook：`window.__gameTest.playtest.captureMiningExpansionGhostPath()`。

### F 移动 + 重建教学

- `computeMobileApproachHint()`：路径不可支付、不可达或无入圈候选时返回教学摘要。
- `resourceGuide` 以 `mining-move-hint` 行展示：「驾驶工站靠近资源外环，再原地重建采矿站」。
- 未修改 `PLAYER_PHYSICS`、移动规则或建造成本。
