## 背景

用户要求 SSDSpaceGame 进入正式大版本交付候选：功能齐全、体验闭环、足以交付给玩家。制作人已批准启动 `v1.0.0-release-candidate` 方向，但当前仍存在采矿收益、P0 路径、自然交火 / 复盘 / Meta、真实输入、小屏触控、发布包与 Release 归档阻塞。

`v0.20.0-mining-coverage-path-evidence` 已作为内部采矿覆盖证据阶段接受，但该阶段只证明诊断能力成立，不代表采矿收益闭环，不代表完整 P0 通过，也不解锁发布准备。

## 严格范围与发布开关

- 严格范围：先完成 1.0 gate，不强行发布。

- 发布、tag、`Releases/` 归档只允许在 #14 制作人最终批准后，按 Git 与发布流程另行执行。

## 风险级别

本版本为高风险版本。以下内容均需专项流程：采矿范围 / 公式、资源布局、移动能力、建造合法性、数值、存档 / Meta schema、发布 tag、`Releases/` 归档。

高风险流程：制作人方向 -> PM 拆分 -> 专项方案 -> 执行自检 -> 专项审核 -> PM 汇总 -> 制作人最终决定 -> 允许后再提交 / 发布。
